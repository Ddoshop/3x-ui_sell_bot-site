# VPN Premium - Веб-сайт

Красивый веб-сайт для управления VPN подписками и активации ваучеров.

## Быстрый старт

```bash
npm install
cp .env.example .env
npm start
```

Сайт будет доступен на `http://localhost:3000`

## Переменные окружения

```env
PORT=3000
NODE_ENV=development
```

## Страницы

### Главная страница (`/`)
Содержит:
- Героическую секцию с описанием
- Список тарифов с ценами
- Форму активации ваучера
- Список особенностей сервиса
- Контактную информацию

### Админ-панель (`/admin`)
Требует авторизацию пароля администратора.

Содержит:
- Дашборд со статистикой
- Управление платежами (подтверждение)
- Управление ваучерами
- Создание новых ваучеров

## Использование

### Для обычного пользователя

#### На главной странице
1. Прокрутите вниз до секции "💎 Выберите подходящий тариф"
2. Выберите нужный тариф нажав "💳 Заказать тариф"
3. Введите ваш Telegram ID (**ВАЖНО**: правильный ID!)
4. Получите реквизиты платежа
5. Оплатите по указанным реквизитам
6. Нажмите "✅ Я отправил платёж"
7. Администратор проверит платёж и отправит ваучер в Telegram

#### Активация ваучера
1. Прокрутите до секции "🎟️ Активировать ваучер"
2. Введите полученный код ваучера
3. Введите ваш Telegram ID
4. Нажмите "✅ Активировать"
5. Подписка будет активирована!

### Для администратора

#### Вход в админ-панель
1. Откройте `http://localhost:3000/admin`
2. Введите пароль администратора (из `.env`)
3. Нажмите "🔓 Вход"

#### Дашборд
Показывает:
- 💰 Общую сумму платежей
- ⏳ Количество ожидающих платежей
- ✅ Количество подтверждённых платежей
- 🎟️ Количество выданных ваучеров

#### Управление платежами
1. Перейдите в раздел "💳 Платежи"
2. Щёлкните на вкладку "⏳ Ожидающие"
3. Нажмите "✅ Подтвердить" для платежа
4. Система автоматически:
   - Создаст ваучер
   - Отправит его пользователю в Telegram
   - Переместит платёж в "✅ Подтверждённые"

#### Создание ваучеров
1. Перейдите в "➕ Создать ваучер"
2. Выберите тариф из списка
3. Укажите количество ваучеров
4. Нажмите "➕ Создать ваучеры"
5. Скопируйте коды кнопкой "📋 Копировать"
6. Распределяйте ваучеры пользователям

## API интеграция

Сайт общается с API на `http://localhost:8788`:

```
GET  /api/plans                      # Получить тарифы
POST /api/payments/create            # Создать платёж
POST /api/vouchers/activate          # Активировать ваучер
GET  /api/users/:userId/subscriptions # Подписки пользователя
```

Админ-панель использует:
```
GET  /api/admin/payments             # Все платежи
POST /api/admin/payments/:id/confirm # Подтвердить платёж
POST /api/admin/vouchers/create      # Создать ваучеры
```
